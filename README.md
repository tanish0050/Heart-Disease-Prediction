# ❤️ Heart Disease Prediction System

This project is a Machine Learning-based system that predicts whether a person is likely to have heart disease based on medical attributes.

## 📌 Features
- Data preprocessing and analysis using Pandas & NumPy  
- Logistic Regression model for classification  
- Train-test split for reliable evaluation  
- Accuracy-based performance measurement  
- Predictive system for real-time input  

## 🧠 Model Details
- Algorithm: Logistic Regression  
- Training Accuracy: ~85%  
- Testing Accuracy: ~82%  

## 📊 Dataset
The dataset contains 303 patient records with 13 medical features such as:
- Age  
- Cholesterol  
- Blood Pressure  
- Heart Rate  
- Chest Pain Type  

## ⚙️ Tech Stack
- Python  
- NumPy  
- Pandas  
- Scikit-learn  

## 🚀 How It Works
1. Load and preprocess data  
2. Split into training & testing sets  
3. Train Logistic Regression model  
4. Evaluate accuracy  
5. Predict heart disease for new input  

## 🎯 Output
- 0 → No Heart Disease  
- 1 → Heart Disease
